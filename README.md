# Адаптивный лендинг для кафе "Уютный уголок"

## О проекте
Современный адаптивный лендинг для небольшого кафе, созданный в рамках учебного задания. Сайт включает интерактивные элементы и полностью адаптирован для всех устройств.

## Стек технологий
- HTML5 с семантической разметкой
- CSS3 с использованием Flexbox/Grid
- JavaScript (чистый, без фреймворков)
- Google Fonts

## Функционал
- Адаптивное меню (бургер-меню для мобильной версии)
- Табы для переключения контента
- Модальное окно для бронирования столика
- Полная адаптивность для десктопа, планшетов и мобильных устройств

## Оптимизация
- Все изображения оптимизированы через Squoosh.app
- Код проверен на валидность в W3C Validator
- Производительность проверена в Lighthouse (оценка 95+)

## Установка и запуск
1. Клонировать репозиторий: `git clone https://github.com/BelleNimos/cafe-landing.git`
2. Перейти в папку проекта: `cd cafe-landing`
3. Открыть файл `index.html` в браузере
